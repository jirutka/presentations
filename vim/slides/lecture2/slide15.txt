
Windows
-------
    Ctrl+W +   (resize, make window bigger, for horizontal split)
    Ctrl+W -   (resize, make window smaller, for horizontal split)
    Ctrl+W =   (the same size)
    Ctrl+W _   (maximize active window)
    Ctrl+W >   (resize, make window bigger, for horizontal split)
    Ctrl+W <   (resize, make window smaller, for horizontal split)
