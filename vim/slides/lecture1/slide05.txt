
Vim modes
---------
▶ Six (VI :) basic modes:
    ◆ Normal
    ◆ Insert
    ◆ Visual
    ◆ Select
    ◆ Command + Commandline
    ◆ Ex (key "Q")
Additional modes:
    ◆ Replace (like Insert)
    ◆ Virtual replace
    ◆ Operator pending mode (motion)
    ◆ Insert Normal Mode (used by evim for example)
    ◆ etc.
